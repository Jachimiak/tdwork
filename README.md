*Notes on AI Use*

Both ChatGPT and Claude were used as tools in the creation of some of the programming. THe process generally worked as follows:
- I would write my initial code. 
- I would test my initial code. If items weren't working exaclty as I had hoped, I would as Claude and/or ChatGPT to help me figure out the solution.
- This iteration would occur multiple times while I worked through the code. Once working, I would test to see if there were any failure points.
- At no time did I have Claude or ChatGPT write the code and use it as it was given to me. Additionally, upon finishing the correc code, I had AI tell me what every line of code did so that I would know how to use create it moving forward.
- Finally, AI can be a tool, but should not be fully trusted for an end product.
